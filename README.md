# ☕ Java Programming Lab Assignments

This repository contains all Java lab assignments completed as part of academic coursework.  
Each experiment demonstrates different core Java concepts including OOP, Exception Handling, Collections, File Handling, JDBC, and JavaFX.

---

## 📚 Experiments Overview

### Experiment 1: Menu-Driven Calculator
- Performs arithmetic operations  
- Uses modular methods  
- Includes input validation  
- Console-based application  

### Experiment 2: Vehicle Management System
- Vehicle class with attributes and methods  
- Manages multiple vehicles using arrays  
- Generates summary reports  

### Experiment 3: Book Inventory System
- Book class with constructors  
- Uses `ArrayList` for record storage  
- Implements exception handling  

### Experiment 4: Vector Operations Program
- 2D/3D vector addition  
- Vector subtraction  
- Dot product calculation  
- Includes exception handling  

### Experiment 5: Banking Application
- Inheritance-based system  
- Different account types  
- Method overriding and polymorphism  

### Experiment 6: Employee Payroll System
- Payroll calculation  
- Demonstrates single, multilevel, and hierarchical inheritance  

### Experiment 7: Student Record Manager
- File-based CRUD operations  
- Uses file handling  
- Includes exception handling  

### Experiment 8: Collections & Design Pattern Application
- Uses Java Collections Framework  
- Implements a basic design pattern  

### Experiment 9: JDBC Database Application
- Database connectivity using JDBC  
- Performs CRUD operations  

### Experiment 10: JavaFX GUI Application
- JavaFX user interface  
- Event handling  
- Backend integration  

---

## 🛠 Technologies Used

- Java (JDK 8 or above)  
- Object-Oriented Programming (OOP)  
- Java Collections Framework  
- File Handling

- JavaFX
